# HTML CSS Bootstrap Task 1

This project is a fully responsive photography contest website developed using HTML5, CSS3, and Bootstrap. The task focused on building a modern and professional user interface by following responsive web design principles and improving frontend development skills.

During this task, I worked on creating a responsive navigation bar, hero banner section, image cards, contest categories, pricing sections, and footer design. Bootstrap grid system and utility classes were used to improve layout responsiveness across desktop, tablet, and mobile devices. Additional custom CSS styling was added to match the reference design accurately and maintain proper spacing, alignment, typography, and hover effects.

The project helped in understanding Bootstrap components, responsive layouts, Flexbox positioning, CSS styling techniques, mobile responsiveness, and GitHub deployment workflow.

## Features

* Fully Responsive Design
* Modern UI Layout
* Bootstrap Grid System
* Mobile Friendly Website
* Hero Banner Section
* Contest Categories Section
* Pricing Cards
* Organized HTML & CSS Structure
* Custom Styling and Responsive Media Queries

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
